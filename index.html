Installing Mods to Minecraft Using Forge

Downloading Forge

In order to instsall mods to Minecraft you have to have a program that will contain all the mod files in the correct location. For this we will use forge.
  • Go to the Forge website.
  • Download the version of forge for the version of Minecraft you want to play on.
  • Once it is downloaded open the Minecraft launcher, there should be a new installation of Minecraft titled "Forge" followed by whatever version you're using.
    • Adding Mods
  • Launch Minecraft using this installation. This will create a mods folder in the Minecraft game files.
  • Press Windows Key + R. This will open a little text box called "Run."
  • In the text bar tpye "%appdata%" (without the quotations). This will bring up a list of folders.
  • Look for a file titled ".minecraft" and double click on it to open. There should be a folder named mods. This is where you are going to put your mods.
  • Now you have a location to put your mods! Now you will just need to download any wanted mods for the correct version of minecraft and then drag-and-drop them into the mods folder.
  • Once you put any desired mods into this folder, you simply launch minecraft on the mods installation and your mods should all be working!
